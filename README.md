A simple blog application using technologies such as:
  - React
  - Redux
  - TypeScript
  - SCSS
  - Bulma

Features:
  - Displaying a list of posts from API
  - Opening a detailed post information
  - Displaying post comments
  - Adding new posts
  - Removing posts
  - Commenting on posts
  
  The repository is self created using create-react-app.
